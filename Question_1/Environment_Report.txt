1. USERNAME AND GROUPS
----------------------------------------------------------------
Commands used:
    $ whoami
    $ id
    $ groups

Output:
    whoami  -> root
    id      -> uid=0(root) gid=0(root) groups=0(root)
    groups  -> root

Observations:
  * whoami returns the effective login name of the account.
  * id shows the numeric UID (0) and GID (0). UID 0 identifies

  * groups lists every group the account belongs to; here the
    only group is 'root'. Group membership controls shared file
    access across the system.
